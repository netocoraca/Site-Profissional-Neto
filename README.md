Este projeto tem como objetivo desenvolver um portfólio profissional para apresentar meus trabalhos nas áreas de modelagem 3D e design/ilustração 2D, além de compartilhar um pouco da minha trajetória e facilitar o contato com potenciais clientes ou recrutadores.

Etapas do Projeto

O desenvolvimento foi dividido em duas fases principais:

1. Idealização do Design

A fase inicial foi dedicada à criação do layout e identidade visual do portfólio.
Ferramenta utilizada:

Adobe Photoshop – para composição, organização estética, escolha de cores, tipografia e estrutura visual das seções.

2. Desenvolvimento do Projeto

Após a definição visual, o projeto foi implementado utilizando:

GitHub – para versionamento, organização do código e publicação.

HTML | CSS | JavaScript – para desenvolvimento da interface e interatividade.
Objetivo do Portfólio

A proposta do portfólio é apresentar meus trabalhos de forma visual, clara e objetiva, destacando:

Projetos de modelagem 3D

Artes e peças como designer 2D / concept

Experimentos e estudos pessoais

Breve explicação da minha trajetória profissional e acadêmica

Informações de contato e links para redes profissionais
